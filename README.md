## Installation

### Prérequis

- Node.js ≥ 24.x
- npm ≥ 11.x

### Cloner le dépôt

```
git clone https://github.com/votre-utilisateur/portfolio.git
cd portfolio
```

### Installer les dépendances

```
npm install
```

## Déploiement

- Connection votre dépôt GitHub à Netfily
- Ajout des variables d'environnement
